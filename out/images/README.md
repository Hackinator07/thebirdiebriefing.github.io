# Image Placeholders

This directory contains images used throughout the Golf Girl Gazette website.

## Image Guidelines

1. **Quality**: Use high-resolution images that look crisp on all devices
2. **Optimization**: Compress images for web use to improve loading speed
3. **Accessibility**: Ensure images have good contrast for text overlay
4. **Branding**: Choose images that align with the Golf Girl Gazette brand and aesthetic

## Alternative Sources

If you need images for the website, consider:
- Stock photo websites (Unsplash, Pexels, Shutterstock)
- LPGA official media resources
- Professional golf photography
- Golf course promotional materials (with permission)
